# Tower of Fantasy Internal
A cheat menu base Tower of Fantasy.
Strictly for educational purposes.

## Credits
* BigbaseV2 Concept by [Pocakking](https://github.com/Pocakking/BigBaseV2)

## Features
* ImGui–based user interface
* Log console
* Thread Pool Concept by [Yimura](https://github.com/Yimura)
* Game Engine Function Access
* Log console
* Script fibers
* Fiber pool

## Building
To build Tower of Fantasy Internal you need:
* Visual Studio 2022
* [Premake 5.0](https://premake.github.io/) in your PATH

To set up the build environment, run the following commands in a terminal:
```dos
git clone https://github.com/skript023/Tower-of-Fantasy.git --recurse-submodules
GenerateProjects.bat
```
Now, you will be able to open the solution, and simply build it in Visual Studio.
